# AT1 – Desenvolvimento de um Sistema Especialista: Recomendação de Músicas

Este documento detalha o planejamento e os requisitos para o desenvolvimento de um Sistema Especialista focado na recomendação musical personalizada, utilizando técnicas de Inteligência Artificial para lidar com a subjetividade e a incerteza.

---

## 🎯 Objetivo
Aplicar os fundamentos históricos e teóricos da IA, utilizando métodos de resolução de problemas e representação do conhecimento para desenvolver um sistema especialista que realize o **tratamento de incertezas** no domínio de recomendações musicais.

## 📚 Conteúdos Abordados
* **História e fundamentos da Inteligência Artificial:** Evolução dos sistemas baseados em regras.
* **Métodos de resolução de problemas:** Estratégias de busca e inferência.
* **Representação do conhecimento:** Mapeamento de preferências e atributos musicais.
* **Tratamento de incerteza:** Aplicação de Lógica Fuzzy para lidar com estados emocionais subjetivos.
* **Introdução à aprendizagem de máquina:** Integração entre abordagens simbólicas (regras) e numéricas.

## 🛠️ Metodologia
O projeto será desenvolvido em etapas pelo grupo:

1.  **Definição do Problema:** Criação de um sistema que recomenda gêneros musicais e ritmos (BPM) com base no humor e nível de energia do usuário.
2.  **Base Teórica e Representação:**
    * Utilização de **Lógica Fuzzy** para traduzir termos vagos (ex: "um pouco triste", "muito animado") em dados processáveis.
    * Definição de **Regras de Produção** (SE entrada ENTÃO saída).
3.  **Desenvolvimento do Protótipo:**
    * Linguagem: **Python**.
    * Bibliotecas: `scikit-fuzzy`, `numpy`.
4.  **Documentação:** Justificativa das escolhas técnicas e análise dos resultados obtidos.
